# Visualizations

This folder contains all data visualizations created for the Social Media Usage & Mental Health Analysis project.

## Files to Upload

Please upload the following visualization files to this folder:

1. **risk_distribution.jpg** - User population distribution by risk level
   - Shows the breakdown of low, medium, and high-risk users
   - Key insight: 53.5% of adolescents require targeted intervention

2. **mental_health_screen_time.jpg** - Mental health outcomes by screen time
   - Line chart showing anxiety and depression rates across screen time categories
   - Key insight: Dramatic escalation at the 4-hour threshold

3. **anxiety_demographics_chart.png** - Anxiety rates across demographics
   - Horizontal bar chart comparing anxiety prevalence
   - Key insight: Gender-neutral problem requiring broad intervention strategies

## Visualization Details

### 1. Risk Distribution Chart
- **Type**: Stacked bar chart
- **Purpose**: Show population segmentation
- **Key Finding**: Majority (53.5%) of users need intervention

### 2. Mental Health vs Screen Time
- **Type**: Dual-axis line chart
- **Purpose**: Demonstrate dose-response relationship
- **Key Finding**: Clear escalation pattern with 4+ hours as critical threshold

### 3. Demographic Comparison
- **Type**: Horizontal bar chart
- **Purpose**: Compare anxiety rates across demographic groups
- **Key Finding**: Minimal gender differences (24.7% vs 23.9%)

## Tools Used

- **Python**: Matplotlib, Seaborn, Plotly
- **Tableau**: Interactive dashboards
- **Design**: Business-focused, presentation-ready
