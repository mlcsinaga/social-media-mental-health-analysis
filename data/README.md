# Data Files

This folder contains all data files used in the Social Media Usage & Mental Health Analysis project.

## Files to Upload

Please upload the following CSV files to this folder:

1. **tableau_dashboard_consolidated_data.csv** - Main dataset (1,500 users)
2. **analyst_executive_summary.csv** - KPI dashboard data
3. **analyst_risk_segmentation.csv** - User segment profiles
4. **analyst_demographic_breakdown.csv** - Demographic analysis
5. **analyst_screen_time_impact.csv** - Dose-response data

## Dataset Description

- **Source**: Synthetic dataset based on CDC adolescent mental health research patterns
- **Size**: 1,500 observations Ã— 17 variables
- **Time Period**: July-December 2024 (6 months)
- **Age Range**: 12-17 years old

### Variables Include:

#### Demographic
- Age, Gender, Urbanization, Parental Education

#### Usage Metrics
- Daily Screen Time, Posts/Day, Comments/Day, Late-Night Posting Ratio

#### Behavioral
- Sentiment Score, Mental Health Subreddit Participation, Isolation Language Frequency

#### Outcomes
- Anxiety Symptoms, Depression Symptoms, Risk Level

#### Derived
- Age Group, Screen Time Category
